Open Source Project Owners!

We want participants to contribute to your projects. The only things we ask is that the project is hosted on Github and that someone will be available during the hackathon to accept pull requests and interact with the participants. Send an email to bryan@opensourcenola.org or tweet @opensourcenola if you want your open source project to be included in the hackathon for users to contribute to.

Participants! Spread the word!

Let your favorite open source projects know so that they can be included in the hackathon

Rules

  • Teams of up to 4 people
  • Anyone on the team can contribute to any open source project(s) listed above
  • Scoring is points based
  • 1 point for submitting a patch or pull request that fixes a bug
  • 2 points for submitting a patch or pull request that implements a feature
  • 4 points creating a new feature, having that accepted by the project and for submitting a patch or pull request implements that feature
  • Whoever scores the most points at the end of the hackathon wins
  • Prizes will be given to the 1st, 2nd, and 3rd place teams
  • Individual prizes will be given to the 1st, 2nd, and 3rd place individuals
